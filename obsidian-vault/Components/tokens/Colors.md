## Code Usage

```css
/* Semantic CSS vars (loaded via tokens.css) */
.screen { background: var(--osmos-bg); }                    /* ScreenBG */
.text-primary { color: var(--osmos-fg); }                   /* Primary Text */
.text-muted { color: var(--osmos-fg-muted); }               /* Secondary Text */
.card { border: 1px solid var(--osmos-border); }            /* Stroke */
.btn-primary { background: var(--osmos-colors-blue-500); }  /* PrimaryButton */
```

```tsx
/* In JSX */
<Box bg="bg" color="fg" borderColor="border" />
<Text color="fg.muted" />
<Button colorPalette="blue" />
```
